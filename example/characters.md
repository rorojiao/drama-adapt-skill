# 人物小传

## 男主：林远

- **年龄**：26岁
- **外貌特征**：身材匀称偏瘦，短发利落，五官端正但不算帅气——属于"丢进人堆找不到"的普通长相。穿着下班时的深色夹克和牛仔裤，运动鞋沾了灰。眼神沉稳，在危急时刻会微微眯眼——这是他集中注意力的习惯。
- **性格核心**：**冷静、果断、隐忍**。遇到危险时不慌不乱，能在极短时间内做出判断。不轻易相信别人，但一旦决定信任就会全力以赴。有正义感但不冲动——救苏晚不是因为"英雄情结"，而是因为隐藏提示告诉他这是最优选择。
- **背景故事**：普通打工人，刚下班走在回家路上就被传送进游戏。没有特殊背景，没有主角光环——除了那个"隐藏提示"技能。正因为普通，他更珍惜活下去的机会。
- **核心动机**：活着离开游戏，搞清楚这个游戏到底是什么、谁在操控。
- **人物弧线**：从独自求生的"精致利己者" → 经历与苏晚的合作 → 开始愿意信任他人、主动分享信息
- **标志性表现**：紧张时会咬牙；做出决定前会短暂沉默；用"隐藏提示"时眼神会快速扫视空中（别人看来像在发呆）
- **与其他角色关系**：
  - 与苏晚：从陌生人到战友，有初步信任和默契。苏晚是他在游戏中第一个信任的人。

## 女主：苏晚

- **年龄**：23岁
- **外貌特征**：杏眼大而明亮，是她最突出的五官特征——恐惧时瞪得圆圆的，笑起来弯成月牙。穿白色卫衣（被传送时穿的），左手臂有一道被行尸抓伤的伤口。头发有些凌乱地扎在脑后，额头有细汗。整体气质是"邻家女孩型"。
- **性格核心**：**勇敢、聪明、信任直觉**。虽然害怕但不会放弃行动——从三楼独自跑到二楼、受伤还能呼救说明她的求生意志极强。愿意相信林远"看到别人看不到的东西"，说明她有开放的心态和判断力。
- **背景故事**：被传送到医院三楼，醒来就遭遇行尸追击。逃跑过程中捡到了医院正门钥匙——这个"幸运"暗示她可能有某种未觉醒的特殊能力。
- **核心动机**：活下去。找到同伴。搞清楚自己为什么会被选入游戏。
- **人物弧线**：从惊恐无助的受害者 → 获救后快速冷静 → 成为能与林远配合的可靠搭档
- **标志性表现**：紧张时抓住身边人的衣角；思考时会咬下唇；有事不绕弯子、直接发问（"你到底是怎么知道这些的？"）
- **与其他角色关系**：
  - 与林远：被他救过命，对他有感激和好奇。是第一个知道林远"能看到隐藏提示"的人。

## 威胁源：行尸

- **类型**：游戏中的基础怪物
- **外貌**：穿着病号服，动作僵硬，脑袋以不自然角度歪着，嘴巴张大发出"嘶嘶"声
- **等级分化**：
  - Lv.1：普通行尸，对声音敏感，弱点后颈，移动缓慢
  - Lv.2：进阶行尸，具有"感知"能力，能探测10米内活物，速度更快
- **在剧中作用**：制造紧张感和生死危机，推动情节发展
