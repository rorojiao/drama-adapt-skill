# 分集大纲

## 总览
- 总集数：6集
- 免费区：第1-3集
- 付费区：第4集起
- 情节弧：醒来困惑 → 初遇危机 → 救援结盟 → 地下室生死追逐 → 极限逃生 → 回到现实发现真相更恐怖

---

## ═══ 免费区（第1-3集）═══

### 第1集：治愈系游戏？
**核心事件**：林远在废弃医院醒来，发现自己被拉入末世求生游戏，首次遭遇行尸
**关键场景**：废弃病房 / 走廊（行尸逼近）/ 病房窗户
**出场人物**：林远、行尸(Lv.1)
**爽点**：隐藏提示首次激活——直接显示行尸弱点和逃生路线，"这技能简直就是作弊！"
**结尾钩子**：林远翻出窗户，站在锈迹斑斑的消防楼梯上，突然听到二楼传来女孩的呼救声——"救……救命……"
**情绪曲线**：困惑 → 恐惧 → 紧张 → 侥幸脱逃 → 新的抉择

### 第2集：铁管英雄
**核心事件**：林远决定救援苏晚，一人干掉两只行尸，获得医院正门钥匙
**关键场景**：消防楼梯 / 二楼病房（战斗）/ 走廊（汇合）
**出场人物**：林远、苏晚、行尸A(Lv.1)、行尸B(Lv.1)
**爽点**：隐藏提示精确分析战斗策略 → 林远完美执行"先B后A"战术 → 一铁管一个，干净利落
**结尾钩子**：隐藏提示解锁地下室秘密路线——"一楼大厅有大量行尸聚集，正面突破极度危险"，唯一活路是黑暗的地下室
**情绪曲线**：犹豫 → 决断 → 紧张战斗 → 胜利快感 → 新危机浮现

### 第3集：太平间的呼吸
**核心事件**：两人潜入地下室太平间，林远冒险拿急救包，惊动Lv.2行尸被追杀
**关键场景**：地下楼梯（黑暗）/ 太平间（冷藏柜）/ 锅炉房铁门前
**出场人物**：林远、苏晚、行尸(Lv.2)
**爽点**：黑暗中凭隐藏提示摸索取宝 → 惊动Lv.2行尸的瞬间肾上腺素飙升 → 铁门关门"嘭"的一声挡住追击
**结尾钩子**：铁门被Lv.2行尸疯狂撞击、剧烈颤抖。隐藏提示冷冰冰地显示："铁门可抵挡约10分钟。"苏晚问："你到底是怎么知道这些的？"林远沉默。
**情绪曲线**：紧张潜行 → 短暂安心 → 冒险 → 极度惊恐 → 绝处逢生 → 悬疑

---

## ═══ 【付费卡点：第3集结尾】 ═══
**卡点设计**：Lv.2行尸正在撞门，铁门只能撑10分钟。苏晚追问林远的秘密。两大悬念同时爆发——他们能逃出去吗？林远会坦白吗？观众在最紧张的时刻被"卡"住，付费意愿最强。

---

## ═══ 付费区（第4-6集）═══

### 第4集：信任
**核心事件**：林远向苏晚坦白"隐藏提示"能力，两人在锅炉房找到通风管道出口
**关键场景**：锅炉房内（治疗+坦白）/ 通风管道口
**出场人物**：林远、苏晚
**爽点**：林远坦白的情感爆发——"如果我说，我能看到别人看不到的东西，你会信吗？"苏晚的回答："我信。在这个疯狂的世界里，什么都有可能。"（第一个信任他的人）
**结尾钩子**：管道出口外，停车场有两只行尸在游荡。倒计时仅剩47分钟。林远说："我们只有一次机会。"
**情绪曲线**：紧迫 → 坦诚 → 温暖 → 新一轮紧张

### 第5集：黎明冲刺
**核心事件**：声东击西战术逃出医院，任务完成，等级提升
**关键场景**：通风管道 / 停车场（声东击西）/ 医院正门
**出场人物**：林远、苏晚、行尸×2(Lv.1)
**爽点**：铁片引开行尸 → 30米冲刺 → 钥匙插入锁孔的颤抖 → 门开的瞬间 → 医院在身后溶解消失 → 升级面板弹出
**结尾钩子**：太阳升起，两人瘫坐。林远说："这只是开始。"面板显示："下一关卡将在24小时后开启……"
**情绪曲线**：紧绷 → 极限冲刺 → 狂喜释放 → 隐忧

### 第6集：游戏……是真的
**核心事件**：回到现实发现游戏系统仍在，苏晚来电揭示玩家社群和第二关60%死亡率
**关键场景**：林远的卧室 / 手机通话
**出场人物**：林远、苏晚（电话）
**爽点**：现实世界中隐藏提示依然有效的震撼 → 苏晚揭示"第二关死亡率60%"的信息炸弹 → 两人积分合体100分的默契
**结尾钩子**：林远看着窗外明媚的阳光，面板显示倒计时18:32:45。"18小时后，第二关就要开始了。"画面定格在林远紧握手机的手上。（强烈暗示续季）
**情绪曲线**：虚假安全感 → 震惊 → 焦虑 → 决心 → 悬念爆满

---

## 结局设计
**结局类型**：开放式结局（第一卷结束，强悬念留续季）
**续季伏笔**：
1. 第二关死亡率60%——比第一关远更危险
2. 神秘的"玩家社群"——背后有什么组织？
3. "好心的前辈"纸条——前辈是谁？还活着吗？
4. 苏晚捡到钥匙的"幸运"——她是否也有未觉醒的特殊技能？
5. 隐藏提示"升级版"预告——新能力是什么？
