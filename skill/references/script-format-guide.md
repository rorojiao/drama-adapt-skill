# 漫剧剧本格式规范参考

## 标准漫剧剧本格式

### 场景头
```
场[集数]-[场次] [日/夜]/[内/外] [场景名称]
```
示例：`场1-1 日/内 豪华办公室`

### 画面描写
- 2-3句简洁描写
- 使用现在时态
- 重点描述视觉元素（光线、色调、构图暗示）
- 适合漫画绘师理解

### 角色对白
```
**[角色名]**
（表情/动作描写）
台词内容。
```

### 画外音/内心独白
```
**OS（[角色名]内心）**：内心独白内容
```

### 闪回
```
**闪回**：[回忆画面描述]
```

### 本集钩子
```
### 【本集钩子】
> [结尾悬念描述]
```

## 漫剧行业标准参数

| 参数 | 标准值 |
|------|--------|
| 单集时长 | 60-120秒 |
| 单集字数 | 500-1000字 |
| 每集场景数 | 3-6场 |
| 每集对白轮次 | 8-15轮 |
| 免费集数 | 前10集 |
| 付费卡点 | 第10集结尾 |
| 爽点密度 | 每2-3集一个主要爽点 |
| 钩子要求 | 每集结尾必有 |

## 题材改编要点

### 都市言情
- 强化情感冲突和误会
- 甜虐交替，3集一个甜蜜高潮
- 付费卡点：通常设在男女主关系重大转折处

### 都市爽文
- 高密度打脸/逆袭情节
- 每1-2集一个爽点
- 付费卡点：通常设在主角获得重大能力/翻盘前

### 玄幻修仙
- 升级体系清晰可追踪
- 战斗场景画面感要强
- 付费卡点：通常设在重大突破/觉醒前

### 悬疑推理
- 线索埋设要前后呼应
- 每集一个小悬念，每5集一个大反转
- 付费卡点：通常设在真相即将揭晓时

### 末世生存
- 紧张感持续递进
- 生死抉择创造共情
- 付费卡点：通常设在重大危机爆发时

## 敏感内容规避

以下内容需要在改编时规避或转化：
- 未成年校园恋爱/暴力
- 涉及体制内的负面描写
- 血腥暴力的详细描写（可用暗示替代）
- 涉毒、涉赌的正面描写
- 迷信、封建糟粕的宣扬
